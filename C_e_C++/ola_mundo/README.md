# C

Compile

Use o gcc

```bash
gcc ola_mundo.c -o ola_mundo.c.o
```

Execute

```bash
./ola_mundo.cpp.o 
```

Use o g++

```bash
g++ ola_mundo.c -o ola_mundo.c.o
```

Execute

```bash
./ola_mundo.cpp.o 
```

# C++

Compile

```bash
g++ ola_mundo.cpp -o ola_mundo.cpp.o
```

Execute

```bash
./ola_mundo.cpp.o 
```

# Tempo de execução 

Compare
C++ vs C

```bash
time ./ola_mundo.cpp.o && time ./ola_mundo.c.o
```

Mude a forma de compilação e compare:
 - C compilado por gcc
