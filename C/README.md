# C

Compile

Use o gcc

```bash
gcc ola_mundo.c -o ola_mundo.c.o
```

Execute

```bash
./ola_mundo.c.o 
```

Use o g++ (Compilador do C++)

```bash
g++ ola_mundo.c -o ola_mundo.c.o
```

Execute

```bash
./ola_mundo.o 
```

# Tempo de execução 

Compare os compiladores: C++ vs C

Mude a forma de compilação:
 - C compilado por gcc
 - C compilado por g++

```bash
time ./ola_mundo.c.o
```
